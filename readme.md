## Proyecto en Laravel 5.6 + React

Para la instalacion debes tener composer y npm.

1- ``composer update``

2- ``npm install``

Para correr el proyecto

1- ``php artisan serve``

Para correr el proyecto en desarrollo y realizar cambios

En una ventana de tu terminal:
1- ``php artisan serve``

Y en otra:

2- ``npm run watch``
